Madera

(desde $58 pesos) La madera es un material muy sólido el cual nos puede servir para poder hacer la base de toda la montaña, para que esta esté estable y nada se mueva. Al igual nos puede servir para poder hacer las columnas que sostienen a los rieles. (este material se puede reciclar y se puede comprar)

Cartón

El cartón es un material sólido, pero este al mojarse se deteriora. Este material nos podría ayudar a hacer prototipos de alguna pieza que queramos hacer y así no gastar material nuevo y no gastar más dinero, ya que este material será reciclado. (reciclado)

Tubos PVC

(desde $20-50) Este material es muy sólido y no se deteriora con el agua y es un material que aguanta muchos golpes y nos podría servir para muchas cosas. Una de ellas sería al hacer las columnas ya que este material aguanta mucho peso y otro seria al hacer los rieles, si estos se parten a la mitad y quedarían las bases de los rieles. (nuevo)

Papel

(desde $58 pesos) El papel nos podría ayudar para hacer prototipos ya que este material es muy frágil no se utilizaría para cosas muy rudas. (reciclado)

Alambre

($32 pesos – 1kg) Se pueden utilizar dos tipos de alambre, uno seria el alambre de los ganchos ya que este tipo de alambre es mucho más grueso y resistente pero un poco difícil de manipular y este material seria reciclado. El otro tipo de alambre seria comprado y este alambre es un poco más delgado y sería más fácil poder manipularlo.

Pegamento UHU

(desde $20-50) Como lo dice es un pegamento que sería comprado, este pegamento es muy bueno y se utiliza para pegar muchas cosas.

Kola loka

($28 pesos) Este pegamento también es muy bueno y seria utilizado para pegar cosas que el resistol UHU no pudiera.
